Filters the elements of the given array from beg\+Index to end\+Index that satisfies the given test and returns a new array. 

This function will take five arguments. It will perform a test given by a function pointed to by test pointer and those elements which return non-\/zero value for test will be stored in the new array. Memory to the new array is allocated from heap. The length of the new array will be stored in the variable pointed to by the length argument. The function will return N\+U\+L\+L under four circumstances-\/ if array is null, pointer to function is null, pointer to length is null and failure to allocate memory. The new array is returned otherwise. Remember to free the memory when done with the new array.
