Simple implementation of filter method.

The elements are filtered on basis of test function. Memory to newArray is allocated from heap. Initially memory is allocated for all the elements. If the length of newArray is less than the original array then the memory is reallocated using realloc which will free up the extra memory. You can use your custom memory allocating technique.
